# ET2100

Repo này chứa các bài tập học phần ET2100 - Cấu trúc dữ liệu và giải thuật, viết chủ yếu bằng C++, bao gồm:
- Tuần 4:
    - Debug
- Tuần 5:
    - Selection Sort
    - Bubble Sort
    - Insertion Sort
- Tuần 6:
    - Quick Sort
    - Merge Sort

## 1. Requirements
- Trình biên dịch C++ (GCC, Clang hoặc MSVC)
- Hỗ trợ C++11 hoặc mới hơn

## 2. How to Run
### Buớc 1: Clone repo
```bash
gh repo clone ThaiHoang026/ET2100_BTVN
```

### Bước 2: Di chuyển đến thư mục chứa file muốn chạy
Ví dụ:
```bash
cd tuan_6
```

### Bước 3: Biên dịch chương trình
Linux/macOS:
```bash
g++ main.cpp sort.cpp -o main
```
Windows:
```bash
g++ main.cpp sort.cpp -o main.exe
```

### Bước 4: Chạy chương trình
Linux/macOS:
```bash
./main
```
Windows:
```bash
./main.exe
```
